# 🧾 Máximo Contabilidade

Website institucional desenvolvido para a empresa **Máximo Contabilidade**, com foco em presença digital profissional, apresentação de serviços e captação de clientes.

---

## 🌐 Acesse o site

👉 https://igor1209.github.io/MaximoContabilidade/

---

## 📌 Sobre o projeto

Este projeto consiste em um site moderno, responsivo e otimizado, criado para apresentar os serviços da empresa de forma clara, profissional e estratégica.

O objetivo principal é fortalecer a presença online da marca, gerar credibilidade e facilitar o contato com potenciais clientes.

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Funcionalidades

- Layout moderno e responsivo (mobile e desktop)  
- Estrutura institucional profissional  
- Seções organizadas (serviços, contato, etc.)  
- Integração com WhatsApp  
- Links clicáveis (localização e redes sociais)  
- Navegação simples e intuitiva
